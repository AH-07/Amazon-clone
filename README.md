# 🛒 Amazon Clone

A front-end Amazon clone built with HTML, CSS, and vanilla JavaScript. Developed as the main project during the [SuperSimpleDev](https://www.youtube.com/@SuperSimpleDev) JavaScript course.

## 🔗 Live Demo

👉 [View it live on GitHub Pages](https://ah-07.github.io/Amazon-clone/)

## Features

- 🔍 Product search by name or keyword
- 🛍️ Add to cart with quantity selector and "Added" confirmation
- 🛒 Checkout page with order and payment summary
- 📦 Orders page showing past orders with dates and totals
- 📬 Package tracking page with a live progress bar (Preparing → Shipped → Delivered)
- 🔄 "Buy it again" button on orders
- Cart quantity counter in the navbar
- Data fetched asynchronously from a backend using `fetch` and `Promise.all`

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Product listing with search and add to cart |
| `checkout.html` | Cart summary and payment breakdown |
| `orders.html` | Order history with product details |
| `tracking.html` | Live delivery progress tracker per product |

## What I Learned

- ES Modules (`import` / `export`) and modular code structure
- `fetch` and `async`/`await` for loading data from a backend
- `Promise.all` for parallel async loading
- URL search params for passing data between pages
- Dynamic HTML rendering from JavaScript data
- Date formatting with the Day.js library
- Progress bar logic using time-based percentage calculations
- `localStorage` for persisting cart data

## Built With

- HTML
- CSS
- JavaScript (ES Modules)
- [Day.js](https://day.js.org/) for date formatting
- Backend API (SuperSimpleDev)
